## 下载
https://oss.chatgo.pro/sync-plugin/obsidian/dinox-sync.zip

## 安装

因为目前插件并没有上传到官方商店，所以需要下载插件解压后手动安装

1. 通过上看的下载链接下载插件
2. 将上面的文件放在你的 vault 目录下的 `.obsidian/plugins` 文件夹中，并解压，然后删除 zip 压缩包
3. 重启 Obsidian 使插件生效

## 使用
1. 获取 token

打开 dinox app 的设置/同步设置/obsidian&思源 点击配置，点击右上角的获取，然后复制 token

2. 填写 token

打开 Obsidian 的配置界面，可以看到社区插件中有 Dinox Sync，点击进行设置，在 Dinox token 中填写第一步获取到的 token

3. 基本设置介绍

    * Dinox directory：如果想要自定义 Dinox 存储的目录，可以自定义路径

    * Filename format: 同步笔记文件的文件名，分别有 time 时间，title 标题，noteId 笔记 ID

    * File layout: 同步笔记文件的布局，nested 表示嵌套结构，会根据天创建文件夹，每天的笔记放入到对应文件夹中，flat 表示平铺，所有笔记直接放到自定义路径下

4. 触发同步

    点击 Obsidain 右下角状态栏中的 dinox 文字，即可触发同步
